# Proyecto-Pel-culas-Sofia-Arias-Yojana-Madera-Dario-Mendez
Código de una aplicación hecha en Python para una agenda de películas con varias opciones para ejecutar.

Integrantes
Sofia Arias Hurtado, Yojana Isabel Madera, Dario Mendez

Opciones de la aplicación

- Consultar la película más larga
- Calcular duración promedio
- Buscar películas
- Encontrar estrenos
- Contar películas 18+
- Reagendar películas
- Verificar si una persona puede ser invitada
- Salir de la aplicación

Archivos para la creación del código

- consola_peliculas.py
- modulo_peliculas.py

Evidencias

Archivo PDF con capturas de pantalla mostrando distintos casos para poder verificar si funciona correctamente
